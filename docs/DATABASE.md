# CareerPilot AI Database Documentation

# Database Overview

CareerPilot AI uses **SQLite** as its primary database.

The database stores user resume information, ATS analyses, AI-generated reports, career recommendations, interview questions, learning roadmaps, analytics, and history.

Database Location

```
storage/database/careerpilot.db
```

---

# Database Architecture

```
Frontend

↓

FastAPI

↓

Repository Layer

↓

SQLAlchemy ORM

↓

SQLite Database
```

---

# Core Data Stored

The database stores the following information:

- Resume Data
- ATS Reports
- Career Reports
- Resume Optimizations
- Job Recommendations
- Interview Questions
- Career Roadmaps
- Dashboard Analytics
- User History

---

# Resume Data

Stores parsed resume information.

Typical fields include:

- Name
- Email
- Phone
- Summary
- Skills
- Education
- Experience
- Projects
- Certifications

---

# ATS Analysis

Stores ATS evaluation results.

Includes:

- ATS Score
- Missing Keywords
- Resume Suggestions
- Analysis Timestamp

---

# Career Reports

Stores AI-generated career reports.

Contains:

- Resume Summary
- Strengths
- Weaknesses
- Skill Gap Analysis
- Career Recommendations
- Learning Resources

---

# Resume Optimizations

Stores optimized resume content generated by AI.

Includes:

- Improved Summary
- Improved Experience
- Better Bullet Points
- ATS Improvements

---

# Job Recommendations

Stores recommended jobs.

Contains:

- Job Title
- Match Percentage
- Required Skills
- Recommendation Reason

---

# Interview Preparation

Stores interview preparation sessions.

Includes:

- HR Questions
- Technical Questions
- Resume-Based Questions

---

# Career Roadmaps

Stores personalized learning roadmaps.

Contains:

- Weekly Goals
- Learning Resources
- Recommended Skills
- Estimated Timeline

---

# Analytics

Stores dashboard statistics.

Examples:

- Total Resumes
- Average ATS Score
- Total Reports Generated
- Total Jobs Recommended

---

# History

Maintains previous AI analyses.

Allows users to revisit:

- ATS Reports
- Career Reports
- Resume Optimizations
- Job Recommendations
- Interview Sessions
- Roadmaps
